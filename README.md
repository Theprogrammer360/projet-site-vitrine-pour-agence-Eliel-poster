# Consulting Elite - Vitrine Digitale

Bienvenue sur le dépôt du projet **Consulting Elite**, une plateforme web moderne conçue pour un cabinet de conseil stratégique haut de gamme.

## 🔗 Lien du projet en direct

👉 [Visiter le site Consulting Elite](https://theprogrammer360.github.io/projet-site-vitrine-pour-agence-Eliel-poster/)

## 📝 Présentation

Ce projet est un site vitrine responsive développé pour offrir une expérience utilisateur fluide et élégante. Il met en avant les services d'expertise en gestion, stratégie financière et transformation digitale.

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique et optimisée.
- **CSS3** : Design moderne avec Flexbox, animations fluides et variables.
- **Responsive Design** : Adapté pour mobiles (iPhone SE, XR, 14 Pro Max), tablettes et PC.
- **GitHub Pages** : Déploiement et hébergement.

## 🚀 Fonctionnalités

- Navigation intuitive et épurée.
- Section Hero avec design asymétrique.
- Grille de services interactive.
- Mise en page fluide grâce aux Media Queries.

_Réalisé par **TheProgrammer360** dans le cadre du développement de l'agence **Elielposter**._
