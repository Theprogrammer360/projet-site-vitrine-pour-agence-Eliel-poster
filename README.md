# Consulting Elite - Vitrine Digitale

Bienvenue sur le dépôt du projet **Consulting Elite**, une plateforme web moderne conçue pour un cabinet de conseil stratégique haut de gamme.

Voir le résultat en direct
Tu peux tester le rendu final ici : [Visiter le site Consulting Elite](https://theprogrammer360.github.io/projet-site-vitrine-pour-agence-Eliel-poster/)

## Présentation

Ce qu'il y a sous le capot
J'ai voulu garder un code propre et efficace :

HTML5 & CSS3 : Pas de frameworks lourds, du pur code pour garder le contrôle total sur le design.

Flexbox : Indispensable pour gérer l'alignement précis de la section Hero et des cartes de services.

Responsive total : Le site a été peaufiné pour être aussi propre sur un iPhone SE que sur un 14 Pro Max ou un écran 4K.

Les points clés
Design "Elite" : Un mélange de bleu marine et de doré pour le côté institutionnel.

Navigation fluide : Une structure pensée pour que le client trouve l'info en deux clics.

Optimisation : Images légères et code indenté pour un chargement rapide.
